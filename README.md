## Publications

### 1. Study of Three Dimensional Modeling of Segregation of Waste and Elimination Using Live Intelligent Machine (SWELIM) Using CAD Software

**Paper ID:** 134

#### Abstract

This research focuses on the development and CAD-based modeling of an automated waste segregation machine named SWELIM. The study involves 3D modeling, structural design considerations, and engineering documentation using professional CAD software.

#### Contributions

### 2. Study of Dimensional Stability of FDM Based 3D Printing of Segregation of Waste and Elimination Using Live Intelligent Machine Components

**Paper ID:** 133

#### Abstract

This study investigates the dimensional stability of FDM-manufactured components used in the SWELIM system. The research evaluates manufacturing accuracy and performance characteristics of 3D-printed parts.

#### Contributions

* Design Evaluation
* Manufacturing Analysis
* Research Documentation
